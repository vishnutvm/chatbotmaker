## ADR template

```markdown

**Status:** proposed | accepted | superseded by NNNN
**Date:** YYYY-MM-DD

## Context
What problem are we solving?

## Options
1. Option A — pros/cons
2. Option B — pros/cons

## Decision
What we chose.

## Consequences
Trade-offs and future implications.
```
